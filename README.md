# 🌟 Maison Étoile

## Descripción
Maison Étoile es un sitio web de lujo y elegancia francesa, diseñado para mostrar servicios de decoración de interiores y productos exclusivos.

## 📁 Estructura del Proyecto
```
maison-etoile/
├── index.html       # Página principal
├── style.css        # Estilos del sitio
├── script.js        # Funcionalidades interactivas
└── README.md        # Este archivo
```

## ✨ Características
- Diseño responsivo (móvil, tablet, escritorio)
- Navegación suave entre secciones
- Galería de inspiración
- Formulario de contacto
- Testimonios de clientes
- Animaciones modernas
- Gradientes elegantes
- Iconos Font Awesome

## 📄 Secciones Incluidas
1. **Navegación** - Barra de navegación fija
2. **Hero** - Sección principal de bienvenida
3. **Sobre Nosotros** - Información de la marca
4. **Servicios** - Tarjetas de servicios principales
5. **Galería** - Inspiración visual
6. **Testimonios** - Opiniones de clientes
7. **Contacto** - Formulario y información de contacto
8. **Footer** - Pie de página con redes sociales

## 🚀 Cómo Usar
1. Descarga todos los archivos
2. Abre `index.html` en tu navegador
3. Explora el sitio web

## 📱 Responsive Design
El sitio está optimizado para:
- Pantallas de escritorio (1200px+)
- Tablets (768px - 1199px)
- Móviles (menos de 768px)

## 🎨 Colores Principales
- Púrpura Gradiente: `#667eea` a `#764ba2`
